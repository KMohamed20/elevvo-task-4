# Task 4: Personal Blog Homepage

> **Level**: Level 2  
> **Requirements Met**: ✅ Blog cards | ✅ Category filter | ✅ Limited posts | ✅ Responsive grid  

## 🔍 Description

A dynamic personal blog homepage displaying posts in a card layout. Users can filter posts by category (**All, Tech, Travel, Food**) using interactive buttons. Only 4 posts are shown initially for performance and UX.

Each post includes:
- Featured image
- Title
- Date
- Short excerpt
- Category badge

## 🖥️ Features

- ✅ **Dynamic Filtering**: Click any category to show only matching posts.
- ✅ **Initial Load Limit**: Only 4 posts rendered at first.
- ✅ **Card-Based Layout**: Clean, modern, hover effects.
- ✅ **Responsive Grid**: Adapts from 3-column desktop to 1-column mobile.
- ✅ **Placeholder Images**: Random images via picsum.photos.

## 🛠️ Technologies Used

| Technology | Purpose |
|----------|---------|
| HTML5    | Post structure and container |
| CSS3     | Card styling, grid layout, hover states |
| JavaScript | DOM manipulation, filtering logic, dynamic rendering |
